Starting a New Project

1. Create a folder for your Project

You can use the command $ mkdr (folder-name) or you can do this through the interface

2. Initilaize Git Repository inside Project

Navigate to the project folder inside of terminal and run the following command:

$ git init

You should see the following output:

Initialized empty Git repository in /home/kayler (user)/Documents (ParentFolder)/day5-3 (project)/.git/

3. Open a browser and go to Github and ceate a new Repository.

Once you cerate a new repository you will copy and paste the folowing commands into terminal. These comands set the Git remote to point to your project folder...

$ git remote add origin https://github.com/(login id / username)/day5.git

As well starting a new branch for you to work on. This branch should be called Master.

$ git push -u origin master

You're all set!

Now read about adding files to your project and pushing your code to Github!
